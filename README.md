# Getting Started

## Prerequisites

[Node.js](https://nodejs.org/en/download) v24.7.0

## Cloning

To clone this repository run:

```
git clone https://github.com/LeapimusMaximus/Matchify-FE
```

## npm packages

To install the npm dependencies run:

```
npm install
```

## Usage

To start a local server run:

```
npm start
```

To run the app in a browser press w.
Otherwise the app can run in [Expo Go](https://expo.dev/go)
